hieuds.com Portfolio Website
============================

A simple professional white-themed portfolio website for Nguyen Ngoc Hieu.

Files:
- index.html: website structure
- style.css: design and layout
- data.js: editable content data
- assets/avatar.png: profile photo
- assets/cv_Hieu.pdf: current CV used in preview
- assets/gallery/: gallery photos
- docs/: future documents, certificates, reports, papers
- HOW_TO_UPDATE.txt: update guide

Open index.html in Chrome to preview locally.
